# Manje Transparent Character Exports

This folder contains app-ready transparent exports of the Manje mascot set.

- Source folder: `../character`
- PNG exports: alpha-channel cutouts, same dimensions and folder structure as the source assets.
- WebP exports: optimized transparent copies for lighter delivery.
- QA: `_qa/transparent-character-contact-sheet.jpg`

The original `assets/character` files are unchanged and remain available as the background-backed production assets.
